toolbar

Methods

  • addButton(opts, index)
参数类型必须说明
optsobject是按钮参数
indexnumber否按钮插入位置索引

opts

参数类型必须说明
namestring否按钮data-name值
classNamestring否按钮className
elHTMLElement否按钮HTML元素，或ZxQuery对象
innerHtmlstring否按钮innerHtml
stylestring否按钮style
eventsarray or object否按钮绑定的事件，单个或多个event数组

events

// object
const evts = {
  name: '',
  type: 'click',
  handler: function(){},
  capture: false
}

zxEditor.toolbar.addButton(evts)

// array
const evtsArr = [
  {
    name: '',
    type: 'click',
    handler: function(){},
    capture: false
  }
]

zxEditor.toolbar.addButton(evtsArr)
参数类型必须说明
namestring否名称
typestring是事件类型
handlerfunction是事件处理程序
captureboolean否默认false
