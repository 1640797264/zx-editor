insertElm

Methods 方法

insertElm

参数：string|HTMLElement|ZxQuery

说明：往编辑器中添加内容。

// string
zxEditor.insertElm('string')

// HTMLElement
const el = document.createElement('div')
el.innerHTML = '<p>This is a paragraph</p>'
zxEditor.insertElm(el)

// ZxQuery
const $el = zxEditor.$('<p>This is a paragraph</p>')
zxEditor.insertElm($el)

$

参数：string|HTMLElement|ZxQuery:any

getText

参数：-

说明：获取编辑器内容。

zxEditor.getText() // 'his is a paragraph'

getHtml

参数：-

说明：获取编辑器内容。

zxEditor.getHtml() // '<p>This is a paragraph</p>'

setHtml

参数：string:html

说明：设置编辑器内容，用户初始化编辑器内容。

zxEditor.setHtml('<p>This is a paragraph</p>')

remove

参数：-

说明：清空编辑器内容。

destroy

参数：-

说明：销毁编辑器DOM及相关事件。

getBase64Images

参数：``

说明：获取编辑器中base64图片数据

返回值：

const result = zxEditor.getBase64Images()
// result
[
  {
    id: 'image id',
    base64: '图片base64数据',
    blob: '图片文件数据，可以用于上传'
  }
]

setImageSrc

zxEditor.setImageSrc(imageID, src)

参数：imageID:图片id，通过getBase64Images获取的id

src: base64（blob）数据上传至服务器后，返回的url地址

修改id为imageID的图片地址。
